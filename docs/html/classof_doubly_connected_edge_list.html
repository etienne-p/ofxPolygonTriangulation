This concept allows for more efficient manipulation of the data structure when splitting face.

Enumerator
None 

No reassignment occurs.

Origin 

The incident edge of the vertex at the origin of the new edge is reassigned.

Destination 

The incident edge of the vertex at the destination of the new edge is reassigned.
